# 1. 알고리즘의 시간 복잡도
* 알고리즘의 소요시간 측정하기
  * 개개인의 컴퓨팅 환경에 따라 **같은 알고리즘이라도 측정 시간이 다르다**.
  * 환경에 영향을 받지 않는 **객관적인 기준**이 필요하다.
  * 객관적인 측정을 위해 알고리즘 내부에서 **기본연산**이 **몇 번** 일어나는지 살펴본다.
  * 성능을 측정할 때는 **입력을 통일**시킨다.
  * 가장 기본연산이 많이 일어나는 **최악의 입력 n개**가 들어온다고 가정한다.
